# Dossier de déploiement Tomcat

Déposez ici votre fichier `.WAR`. Tomcat le déploie automatiquement (auto-deploy).

- `monappli.war` → accessible sur http://localhost:8888/monappli/
- `ROOT.war`     → accessible sur http://localhost:8888/

Ce dossier est monté dans le conteneur sur `/usr/local/tomcat/webapps`.
